## Задача 7
Написать программу на С: открыть файл (произвольного размера без
ограничений), доступный для чтения. Имя файла передается из командной строки. После
открытия файла создать параллельный процесс с помощью fork. В обоих процессах
создать свои файлы для записи, читать информацию из общего файла и копировать ее в
собственные выходные файлы (не на экран). Вывести на экран содержимое полученных
файлов из порожденного процесса по окончании записи в файл и из родительского
процесса, дождавшись завершения порожденного процесса.

#### Сборка (находясь в директории задания):
```bash
make clean && make
```

#### Пример запуска:

```bash
./build/main.bin input.txt
```
