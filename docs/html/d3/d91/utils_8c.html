Detailed Description

Реализация вспомогательных функций для работы со строками в общем буфере.

Файл содержит реализацию функций, необходимых для корректной работы алгоритмов сортировки. Эти функции позволяют обрабатывать строки без изменения исходного порядка в памяти.
