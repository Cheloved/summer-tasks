Detailed Description

Реализация алгоритма быстрой сортировки строк.

Файл содержит реализацию функции quick_sort(), которая рекурсивно сортирует строки по их индексам, используя разделение на подмассивы относительно опорного элемента. Для сравнения используется linecmp().

Быстрая сортировка массива индексов строк.

Рекурсивно разбивает массив на три части (меньше, равно, больше опорного), сортирует подмассивы и возвращает отсортированный массив индексов строк.
