Detailed Description

Реализация многопоточной сортировки слиянием (Parallel Merge Sort)

В данном файле реализована многопоточная версия алгоритма сортировки слиянием, позволяющая использовать несколько потоков для ускорения процесса сортировки. Алгоритм сохраняет временную сложность O(n log n) и использует библиотеку pthreads для распараллеливания задач на этапах разделения массива.

Author
Ракитин Илья Алексеевич
Date
29.04.2025

Реализация использует стандартную библиотеку <pthread.h> для создания параллельных потоков. Merge-операция остаётся последовательной из-за необходимости сохранить стабильность сортировки.
