Находит самую длинную возрастающую подпоследовательность с наибольшей суммой.

Функция анализирует входную последовательность целых чисел и находит подпоследовательность, которая:

  • строго возрастает,
  • имеет наибольшую сумму (если есть несколько таких — выбирается самая длинная).

Результат записывается в выходной массив result, а длина результата — в result_size.

Parameters
seqВходная последовательность целых чисел.
sizeДлина входной последовательности.
resultВыходной массив, в который будет записан результат.
result_sizeУказатель на переменную, в которую будет записана длина результата.
Returns
Возвращает 0 при успешном завершении.
