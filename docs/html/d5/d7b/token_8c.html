Detailed Description

Реализация функции лексического анализа строки.

Содержит реализацию функции tokenize(), которая разбивает строку на отдельные токены, используя разделители — пробелы и скобки.

int tokenize ( char * input,
char ** output )

Разбирает входную строку на токены.

Функция разделяет строку на логические элементы: числа, операции, скобки.

Parameters
inputВходная строка, содержащая выражение.
outputВыходной массив для хранения токенов. Должен быть размером MAX_TOKENS x MAX_TOKEN_LEN.
Returns
Возвращает количество найденных токенов или 0 при ошибке.
