Detailed Description

Реализация параллельной сортировки слиянием с использованием потоков.

Реализует функцию merge_sort(), которая запускает сортировку в отдельном потоке. Каждый рекурсивный шаг делится между потоками. После окончания сортировки частичные результаты объединяются.
