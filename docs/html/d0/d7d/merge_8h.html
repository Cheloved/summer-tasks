Detailed Description

Заголовочный файл реализации параллельной сортировки слиянием.

Содержит объявление структуры данных для передачи параметров в поток, объявления функций для запуска сортировки

Объединяет два отсортированных массива в один.

Сравнивает элементы по значениям строк через функцию linecmp(). Сохраняет результат в поле result структуры data.
