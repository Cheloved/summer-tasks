Detailed Description

Реализация функций для работы с квадратными матрицами.

Файл содержит реализацию базовых операций над квадратными матрицами:

  • выделение памяти,
  • умножение матриц,
  • вывод матрицы,
  • освобождение памяти.

Используются глобальные переменные global_row и global_col для отслеживания позиции во время умножения (например, для логирования или отладки).
