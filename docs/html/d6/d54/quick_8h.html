Detailed Description

Заголовочный файл для реализации быстрой сортировки строк.

Содержит объявление функции quick_sort(), которая выполняет рекурсивную быструю сортировку индексов строк текста, используя сравнение через linecmp().

Быстрая сортировка массива индексов строк.

Рекурсивно разбивает массив на три части (меньше, равно, больше опорного), сортирует подмассивы и возвращает отсортированный массив индексов строк.

Parameters

lengthДлина входного массива индексов.
resultВыходной массив отсортированных индексов строк.
Returns
Возвращает 0 при успешном завершении.
