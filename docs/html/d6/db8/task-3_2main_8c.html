Detailed Description

Точка входа программы сортировки строк текстового файла.

Программа принимает на вход имя входного файла, имя выходного файла и метод сортировки. Поддерживаются следующие методы: пузырьковая (bubble), быстрая (quick) и сортировка слиянием (merge).

Главная функция программы.

Проверяет количество аргументов командной строки. Если их недостаточно, вызывает print_help() и завершает выполнение. Иначе вызывает функцию sort_file(), передавая ей пути к входному и выходному файлам, а также метод сортировки.
