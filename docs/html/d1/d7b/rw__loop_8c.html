Detailed Description

Реализация функции read_loop() для работы с циклом чтения/записи через UNIX-сокет.

Файл содержит реализацию функции, которая создаёт два процесса: один читает данные из сокета, второй — отправляет введённые пользователем данные в сокет. Обмен происходит до разрыва соединения.

Разделяет чтение и запись в сокет на два отдельных процесса.

Создаёт дочерний процесс: родительский процесс ожидает входящие данные из сокета, дочерний — считывает ввод с терминала и отправляет его через сокет. Работает до закрытия соединения одной из сторон.

Parameters
sockfdДескриптор сокета, через который осуществляется обмен данными.
connect_nameИмя соединения (например, "client" или "server") для логирования.
Returns
Возвращает 0 при успешном завершении, -1 в случае ошибки.
