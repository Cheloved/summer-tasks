Detailed Description

Заголовочный файл для реализации пузырьковой сортировки строк.

Содержит объявление функции bubble_sort(), которая выполняет пузырьковую сортировку индексов строк текста, находящегося в общей памяти.

Пузырьковая сортировка массива индексов строк.

Сортирует массив индексов строк на основе содержимого текста, используя функцию linecmp() из utils.h для сравнения строк.

Parameters

resultВыходной массив отсортированных индексов строк.
Returns
Возвращает 0 при успешном завершении.
