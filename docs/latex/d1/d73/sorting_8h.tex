\doxysubsection{Detailed Description}
Общие функции для работы с текстовыми файлами и управления сортировкой строк. 

Содержит объявления функций, необходимых для чтения файла, подсчёта строк, получения индексов строк, выбора метода сортировки и сохранения результата.
