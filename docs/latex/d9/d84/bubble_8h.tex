\doxysubsection{Detailed Description}
Заголовочный файл для реализации пузырьковой сортировки строк. 

Содержит объявление функции {\ttfamily \doxylink{bubble_8h_aba3baf667f64d13cd436e62f3d0af30f}{bubble\+\_\+sort()}}, которая выполняет пузырьковую сортировку индексов строк текста, находящегося в общей памяти.
