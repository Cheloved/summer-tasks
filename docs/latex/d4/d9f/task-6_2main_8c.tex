\doxysubsection{Detailed Description}
Пример использования системного вызова stat() для получения информации о файле. 

Программа принимает в командной строке пути к файлам и выводит подробную информацию о каждом\+:
\begin{DoxyItemize}
\item тип файла,
\item права доступа,
\end{DoxyItemize}

размер,
\begin{DoxyItemize}
\item временные метки (доступ, изменение, изменение статуса),
\item UID/\+GID владельца и др.

\begin{DoxyAuthor}{Author}
Ракитин Илья Алексеевич 
\end{DoxyAuthor}

\end{DoxyItemize}
