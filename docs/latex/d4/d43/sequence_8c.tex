Находит самую длинную возрастающую подпоследовательность с наибольшей суммой. 

Функция анализирует входную последовательность целых чисел и находит подпоследовательность, которая\+:
\begin{DoxyItemize}
\item строго возрастает,
\item имеет наибольшую сумму (если есть несколько таких — выбирается самая длинная).
\end{DoxyItemize}

Результат записывается в выходной массив {\ttfamily result}, а длина результата — в {\ttfamily result\+\_\+size}.
