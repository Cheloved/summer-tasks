\doxysubsection{Detailed Description}
Реализация парсера логических выражений. 

Файл содержит реализацию следующих этапов обработки выражения\+:
\begin{DoxyEnumerate}
\item Разбор на токены (лексический анализ),
\item Преобразование в постфиксную форму (алгоритм Дейкстры),
\item Вычисление выражения с интерактивным вводом значений переменных.
\end{DoxyEnumerate}
