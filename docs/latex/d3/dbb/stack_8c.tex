\doxysubsection{Detailed Description}
Реализация стека строк. 

Файл содержит реализацию базовых операций над стеком строк\+:
\begin{DoxyItemize}
\item проверка на пустоту и заполненность,
\item добавление элемента (push),
\item извлечение элемента (pop),
\item просмотр верхнего элемента (peek).
\end{DoxyItemize}
