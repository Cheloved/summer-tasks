\doxysubsection{Detailed Description}
Реализация алгоритма быстрой сортировки строк. 

Файл содержит реализацию функции {\ttfamily \doxylink{quick_8c_aaba19a5bf0aad600311eb39ac355c940}{quick\+\_\+sort()}}, которая рекурсивно сортирует строки по их индексам, используя разделение на подмассивы относительно опорного элемента. Для сравнения используется \doxylink{utils_8c_a3a3b23666dbdc2c1d11be1e20d20d9b9}{linecmp()}.

Быстрая сортировка массива индексов строк.
