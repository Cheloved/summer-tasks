\doxysubsection{Detailed Description}
Заголовочный файл, реализующий стек строк. 

Содержит объявление структуры {\ttfamily \doxylink{structs__stack}{s\+\_\+stack}} и функций для работы со стеком\+:
\begin{DoxyItemize}
\item проверка на пустоту/переполнение,
\item добавление элемента (push),
\item извлечение элемента (pop),
\item просмотр верхнего элемента (peek).
\end{DoxyItemize}
