\doxysubsection{Detailed Description}
Пример работы с обработкой сигналов и умножением матриц. 

Программа\+:
\begin{DoxyItemize}
\item позволяет выбрать способ обработки сигнала {\ttfamily SIGINT}\+: через {\ttfamily signal()} или {\ttfamily sigaction()},
\item считывает две матрицы из файлов,
\item перемножает их,
\item завершает работу по сигналу {\ttfamily Ctrl+C} после вывода текущих координат вычисления.
\end{DoxyItemize}

Также поддерживается режим генерации случайных матриц в файлы {\ttfamily a.\+txt} и {\ttfamily b.\+txt}.
