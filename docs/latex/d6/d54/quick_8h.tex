\doxysubsection{Detailed Description}
Заголовочный файл для реализации быстрой сортировки строк. 

Содержит объявление функции {\ttfamily \doxylink{quick_8h_aaba19a5bf0aad600311eb39ac355c940}{quick\+\_\+sort()}}, которая выполняет рекурсивную быструю сортировку индексов строк текста, используя сравнение через \doxylink{utils_8c_a3a3b23666dbdc2c1d11be1e20d20d9b9}{linecmp()}.
