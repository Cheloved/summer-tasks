\doxysubsection{Detailed Description}
Точка входа программы для анализа числовой последовательности. 

Программа\+:
\begin{DoxyItemize}
\item принимает путь к файлу и количество элементов,
\item считывает последовательность из файла,
\item находит самую длинную возрастающую подпоследовательность с наибольшей суммой,
\item выводит результат на экран.
\end{DoxyItemize}
