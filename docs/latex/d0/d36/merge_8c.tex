\doxysubsection{Detailed Description}
Реализация параллельной сортировки слиянием с использованием потоков. 

Реализует функцию {\ttfamily \doxylink{merge_8c_ae1a8323308a85218b2650c63a3b560d4}{merge\+\_\+sort()}}, которая запускает сортировку в отдельном потоке. Каждый рекурсивный шаг делится между потоками. После окончания сортировки частичные результаты объединяются.
