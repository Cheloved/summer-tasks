\doxysubsection{Detailed Description}
Точка входа программы, реализующей парсер и вычислитель логических выражений. 

Программа\+:
\begin{DoxyItemize}
\item принимает строку с логическим выражением,
\item разбивает её на токены,
\item преобразует в постфиксную запись (обратная польская),
\item запрашивает значения переменных,
\item вычисляет результат выражения.
\end{DoxyItemize}

Поддерживаемые операторы\+: {\ttfamily NOT}, {\ttfamily AND}, {\ttfamily OR}.
