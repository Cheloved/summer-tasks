\doxysubsection{Detailed Description}
Заголовочный файл для работы с квадратными матрицами. 

Содержит объявления функций для инициализации, умножения, вывода и освобождения памяти динамических квадратных матриц. Также содержит глобальные переменные, используемые в процессе вычислений.
