\doxysubsection{Detailed Description}
Пример использования системного вызова fork() и копирования содержимого файла. 

Программа\+:
\begin{DoxyItemize}
\item принимает путь к файлу в аргументах командной строки,
\item запускает дочерний процесс через {\ttfamily fork()},
\item оба процесса (родительский и дочерний) читают файл и записывают его содержимое в разные выходные файлы,
\item затем выводят это содержимое на экран.
\end{DoxyItemize}
