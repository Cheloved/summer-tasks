\doxysubsection{Detailed Description}
Реализация алгоритма пузырьковой сортировки строк. 

Файл содержит реализацию функции {\ttfamily \doxylink{bubble_8c_aba3baf667f64d13cd436e62f3d0af30f}{bubble\+\_\+sort()}}, которая реализует пузырьковый метод сортировки строк по их индексам. Используется для сортировки строк текстового файла без изменения оригинального порядка в памяти.
