\doxysubsection{Detailed Description}
Реализация функций для работы с квадратными матрицами. 

Файл содержит реализацию базовых операций над квадратными матрицами\+:
\begin{DoxyItemize}
\item выделение памяти,
\item умножение матриц,
\item вывод матрицы,
\item освобождение памяти.
\end{DoxyItemize}

Используются глобальные переменные {\ttfamily global\+\_\+row} и {\ttfamily global\+\_\+col} для отслеживания позиции во время умножения (например, для логирования или отладки).
