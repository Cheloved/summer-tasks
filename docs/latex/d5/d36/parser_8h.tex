\doxysubsection{Detailed Description}
Заголовочный файл для парсинга и вычисления логических выражений. 

Содержит объявления функций, необходимых для\+:
\begin{DoxyItemize}
\item проверки является ли строка оператором,
\item преобразования инфиксного выражения в постфиксное (обратная польская запись),
\item вычисления значения выражения.
\end{DoxyItemize}

Поддерживаются операторы\+: NOT, AND, OR.
