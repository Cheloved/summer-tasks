\doxysubsection{Detailed Description}
Вспомогательные функции для работы со строками в общем буфере. 

Содержит объявления функций, используемых алгоритмами сортировки для\+:
\begin{DoxyItemize}
\item получения длины строки по индексу,
\item сравнения двух строк по их позиции в буфере,
\item конкатенации строк по массиву индексов.
\end{DoxyItemize}
