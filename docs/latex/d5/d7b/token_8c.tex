Реализация функции лексического анализа строки. 

Содержит реализацию функции {\ttfamily \doxylink{token_8c_aa70ffa26ede49f51cdffb00ecf5c10b6}{tokenize()}}, которая разбивает строку на отдельные токены, используя разделители — пробелы и скобки.
